# TuDistri Frontend Source

Código fuente de la aplicación React.

## Estructura de Directorios

*   **`api/`**: Cliente HTTP autogenerado. Conecta con el Backend.
*   **`components/`**: Componentes UI reutilizables (Botones, Mapas, Gráficos).
*   **`hooks/`**: Custom Hooks para lógica de estado compleja (ej: controlador de simulación).
*   **`pages/`**: Vistas principales (Ruteo).
*   **`services/`**: Configuración de servicios externos (Axios instances).
*   **`store/`**: Gestión de estado global con Zustand.
*   **`utils/`**: Funciones auxiliares (formateo de moneda, fechas).
*   **`App.tsx`**: Componente raíz y configuración de Router.
*   **`main.tsx`**: Punto de montaje en el DOM.
