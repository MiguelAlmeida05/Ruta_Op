# UI Components

Componentes visuales de la aplicación.

## Componentes Clave

*   **`DarkMap.tsx`**: Mapa base interactivo usando `react-leaflet`. Soporta modo oscuro y renderizado de rutas.
*   **`MapLayers.tsx`**: Maneja capas superpuestas en el mapa (Rutas, Marcadores de Vendedores, Tráfico).
*   **`ProductSimulator.tsx`**: Panel de control para seleccionar productos e iniciar simulaciones.
*   **`MetricsDashboard.tsx`**: Visualización de KPIs (Gráficos de barras/líneas con Recharts).
*   **`VehicleMarker.tsx`**: Icono animado o estático para representar la ubicación del usuario/vehículo.
