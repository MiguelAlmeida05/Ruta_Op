# Backend Scripts

Scripts de utilidad para mantenimiento y operaciones DevOps.

## Archivos

*   **`update_data.py`**: Fuerza la actualización de los grafos y datos estáticos.
*   **`benchmark_graph_libs.py`**: Compara rendimiento entre `networkx` y `rustworkx`.
*   **`seed_history.py`**: Genera datos históricos falsos en Supabase para demos.
*   **`train_eta_model_mock.py`**: Entrena un modelo XGBoost básico para pruebas de ETA.
*   **`debug_rx.py`**: Script para depurar problemas de conversión de grafos.
