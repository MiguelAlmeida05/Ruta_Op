# Services Module

Este módulo contiene la lógica de negocio "pura" y los adaptadores tecnológicos.

## Subdirectorios

*   **`graph/`**: Gestión de carga y procesamiento de grafos (NetworkX/OSMnx).
*   **`routing/`**: Algoritmos de búsqueda de caminos.
*   **`simulation/`**: Motor estocástico de simulación logística.
*   **`validation/`**: Servicios para verificar la consistencia y estabilidad de los resultados.
